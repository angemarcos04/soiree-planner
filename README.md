# Soirée Planner

A beautiful event planning application built with JavaScript, CSS, and HTML.

## Project Overview

Soirée Planner is a modern, user-friendly application designed to help you organize and manage your events efficiently. With an intuitive interface and comprehensive features, planning your next soirée has never been easier.

## Tech Stack

- **HTML** - 3.4%
- **CSS** - 50.9%
- **JavaScript** - 45.7%
